## Решения алгоритмов на Rust
Rust — современный системный язык программирования с производительностью, схожей с C/C++, и возможностями низкоуровневой оптимизации, что делает его отличным выбором для написания алгоритмов и решения задач LeetCode.

`algos` содержит реализации различных распространённых алгоритмов и структур данных, а также решения задач LeetCode. Для соответствующих алгоритмов предоставлена документация и комментарии на русском языке, чтобы помочь вам учиться быстрее и эффективнее.

> Данный репозиторий является переводом [оригинального китайского учебника](https://github.com/rustcn-org/rust-algos) с китайского языка на русский.

## Изучение Rust
Если вы знакомы с алгоритмами, но недостаточно знаете Rust, обратитесь к [Rust Language Bible](https://github.com/sunface/rust-course) — это один из лучших русскоязычных открытых учебников по Rust.
