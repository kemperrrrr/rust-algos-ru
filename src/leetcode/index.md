# Решения LeetCode

В настоящее время в интернете относительно мало решений задач LeetCode на Rust, и ещё меньше — с подробными объяснениями. В этом учебнике мы стремимся изменить эту ситуацию и написать понятные каждому решения LeetCode. Планируется разбирать задачи по порядку номеров — один день, одна задача, по возможности с подробным описанием каждого шага. Если вы заметите недостатки, сообщите нам или отправьте PR.
